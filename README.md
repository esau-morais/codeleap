# CodeLeap Network

Social network application built with React 19, TypeScript, and Tailwind CSS 4.

## Features

- CRUD operations for posts
- Real-time collaboration (PartyKit WebSocket)
- Live user presence counter
- Username-based authentication (localStorage)
- Infinite scroll pagination
- Search and date filtering
- Draft autosave
- Time-based post grouping
- Character counters
- Toast notifications

## Tech Stack

- React 19 + TypeScript
- Vite
- Tailwind CSS 4
- TanStack Query
- React Hook Form + Zod
- Zustand
- PartyKit (WebSocket)
- Sonner
- Biome

## Development

```bash
bun install
bun run dev
```

## Build

```bash
bun run build
bun run preview
```
